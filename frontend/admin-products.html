Products are managed in JavaScript

Your products are stored in frontend/js/products.js as a JavaScript object. This makes them easy to manage and doesn't require database storage.

To add/edit products: Open products.js and add new product objects with details like name, price, images, etc.

Example product structure:

'product-id': {
    name: 'Product Name',
    category: 'helium',
    image: 'images/product.jpg',
    standardPrice: 350,
    hifloatPrice: 400,
    description: [...],
    delivery: [...]
}
